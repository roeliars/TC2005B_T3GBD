Dashboard

Propósito

Conocer cómo usar los datos que tenemos en nuestra DB para crear gráficas en nuestras páginas web.

Instrucciones

Crear un nuevo proyecto de Django. El proyecto deberá tener una aplicación y la aplicación deberá tener 1 página web que muestre al menos una gráfica cuyos datos provengan de la DB y al menos un dato simple y un listado de datos.


Requerimientos generales:

  1. Crear un proyecto de Django
  2. Crear una aplicación en el proyecto.
  3. Crear un modelo de DB (sqlite3) que contenga al menos 1 tabla. Se sugiere que tomen como inspiración alguna de las tablas de su modelo del reto. Quizá la mejor tabla para comenzar a trabajar es aquella donde guardamos las interacciones de los estudiantes con nuestro videojuego. Esta tabla debería tener alguna información sobre las acciones, tiempos y resultados que los estudiantes obtienen al interactuar con el juego.
  4. Crear una página web en html que muestre una gráfica con algunos de los datos de la tabla. Además se deberá mostrar en otra parte de la página algún dato individual que se haya obtenido de la misma tabla. Finalmente se deberá desplegar un listado de algunos de los registros provenientes de la misma tabla en formato de tabla o lista.
